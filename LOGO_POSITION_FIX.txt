Install in Termux:
1. Stop the server with Ctrl + C.
2. cd ~
3. unzip -o ~/storage/downloads/Hamilton_International_Hotel_HMS_Logo_Position_Fix.zip
4. cd ~/hamilton-smart-hms
5. npm start
6. Open http://localhost:8080/app.html
